# Projetos-Curso-T-cnico-Osvaldo-Cruz

## Dúvidas Frequentes:
### ❓Como posso achar meus projetos?
O caminho das pastas funciona da seguinte forma: Série/Matéria/Projeto (Ex: SegundoAno/ProgramacaoWeb/FrutifousApp)

### ❓Não faço parte do curso técnico/já me formei, posso baixar esses projetos?
Sim! Esses projetos são feitos para fins educacionais e não estão relacionados à quaisquer fins comerciais (se possível, só peço que dê os devidos créditos)

### ❓Posso baixar o repositório todo de uma vez?
Ao baixar o repositório, você terá localmente todos os projetos dele. Apesar de não serem projetos que exigem muito armazenamento (por serem simples), recomendo baixar somente os que você deseja
