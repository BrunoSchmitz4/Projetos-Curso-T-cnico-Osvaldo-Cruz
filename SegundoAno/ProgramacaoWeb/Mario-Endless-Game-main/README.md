About the Project
This project is a simple endless runner game featuring the classic Mario character, built using standard web technologies: HTML, CSS, and JavaScript.

The primary goal of this project was to learn and practice web game development. Since I am more familiar with web development environments, I chose this stack to understand how to apply web skills (DOM manipulation, styling, and basic JavaScript logic) to create an interactive gaming experience.

## Gameplay
The objective is simple: Mario must jump over an endless series of pipes that periodically appear on the screen.

The player gains points for every pipe successfully dodged.

If Mario collides with a pipe, the game ends.

The final score is displayed upon losing.

## Learning Focus
Through this game, I focused on core concepts of 2D web game development, including:

- DOM Manipulation: Creating, moving, and removing game elements (Mario, pipes) using JavaScript.

- CSS Animations: Handling the movement of the background and the appearance of pipes.

- Game Loop Logic: Implementing periodic generation of obstacles and collision detection.

- State Management: Tracking the score and managing the game's start/end states.

🛠️ Technologies Used
- Markup: HTML5

- Styling: CSS3

- Logic: JavaScript (Vanilla JS)
